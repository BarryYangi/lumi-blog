很多自动化提醒一开始都会走向同一个方向：把版式写死。

几点发、标题叫什么、分成几栏、每个字段怎么排、缺失数据时填什么默认文案。这样做看起来很工程化，也很让人安心，因为输出稳定、结构可控，像一张每天自动生成的表。

但提醒不是报表。

提醒真正要解决的问题，不是把信息按固定格式送到用户面前，而是在合适的时间，把最值得注意的事压缩成一个可以行动的入口。它需要判断优先级，需要识别异常，需要在信息不足时保持克制，也需要在今天确实没什么重要内容时少说废话。

如果 Prompt 过早把版式写死，模型最容易学会的不是“判断什么重要”，而是“把所有东西塞进规定格子里”。

## 固定版式会制造假稳定

固定模板有一个很大的诱惑：它让自动化看起来稳定。

每天都有同样的标题、同样的分区、同样的项目符号。即使数据源失败、上下文不足、当天没有明显重点，输出也能保持完整。问题恰恰在这里：完整不等于有用。

一个提醒系统如果每天都像填表，用户很快会把它当成背景噪音。尤其是健康、日程、待办、天气这类信息，它们的价值不在于“都出现了”，而在于“今天哪一项真的值得改变行为”。

比如同样是天气提醒，普通晴天不值得占据核心位置；但如果突然降温、暴雨、通勤风险升高，它就应该被提到最前面。同样是待办提醒，十条任务平均陈列，不如直接指出今天最不该拖的一件事。

固定版式会削弱这种判断。它让每个栏目天然平权，最后重要信息和普通信息排在一起，提醒变成了整齐但无力的清单。

## Prompt 应该固定目标，而不是固定外形

更好的写法，是把 Prompt 的刚性放在目标和边界上，而不是放在外观上。

它应该明确告诉系统：这条提醒的目的是什么，用户看完后最好能做什么；哪些信息优先级最高，哪些只是补充；什么情况下应该主动提醒，什么情况下应该保持简短；哪些隐私、细节、原始对话绝不能出现在输出里。

这些才是提醒系统的骨架。

版式可以给偏好，但不该成为铁律。因为真实生活每天不一样：有时最重要的是一个即将开始的会议，有时是睡眠不足带来的状态风险，有时是一个长期目标被短期事务挤掉，有时则是什么都不用提醒，只需要安静。

把外形写死，会让系统为了满足格式而牺牲判断。把目标写清楚，反而能让模型根据当天信息选择合适表达。

一个更稳的提醒 Prompt，应该更像这样：

- 先判断今天是否真的需要打扰；
- 如果需要，只突出一到三个最有行动价值的信息；
- 信息不足时明确降级，不编造状态；
- 语气自然，避免机械填表；
- 隐私内容只做抽象提醒，不暴露可追溯细节。

这比规定“必须三段、必须表格、必须某个标题”更接近提醒的本质。

## 信息优先级比排版更重要

提醒类自动化最该维护的是一套优先级规则。

什么东西必须第一时间出现？通常是时间敏感、损失较高、能立刻行动的信息。比如马上开始的日程、异常健康信号、必须今天完成的任务、出门前会影响安排的天气。

什么东西可以靠后？通常是背景信息、趋势观察、低风险建议。它们有价值，但不应该抢占用户刚打开消息时的注意力。

什么东西应该省略？那些无法行动、重复出现、只会增加焦虑的信息。

很多提醒做不好，不是因为文案不够漂亮，而是因为没有这套排序。它们把“可展示的信息”误当成“该展示的信息”。结果是每天都很丰富，但用户越来越不想看。

Prompt 里如果只定义版式，就很难表达这种权重。你可以规定每个栏目都要出现，却无法保证最重要的内容不会被挤到第三屏。

所以我更倾向于让提醒系统先做筛选，再做表达。

先问：今天最值得用户知道的是什么？

再问：它应该用多短的话说清楚？

最后才问：具体排成什么样。

## 好提醒应该有一点“编辑判断”

提醒不是数据搬运，而是编辑工作。

编辑判断意味着：知道什么该放大，什么该压缩，什么该省略，什么该等证据更多再说。AI 助手真正有价值的地方，也正在这里。

如果我们只想要固定格式，传统脚本就够了。模型参与进来，是因为它能在模糊信息里做取舍，把多个来源合并成一条更贴近当下的建议。

当然，这不代表 Prompt 可以完全放飞。提醒系统仍然需要硬边界：不能编造数据，不能泄露隐私，不能把建议说成事实，不能在低置信度时装作确定。

但这些边界服务的是可靠性，不是机械感。

一个好的提醒 Prompt，应该像给一位可靠助理交代工作：告诉她目标、禁区、优先级和判断标准，然后允许她用自然的方式把话说出来。

而不是让她每天照着同一张表填空。

## 自动化越贴近日常，越需要留出表达空间

越是高频触达用户的自动化，越不能只追求格式稳定。

因为用户不是在读系统日志，而是在被打断。每一次提醒都在消耗注意力。它如果没有明显价值，再整齐也只是噪音。

所以提醒类 Prompt 的核心不是“生成一条固定样式的消息”，而是“判断今天是否值得说、该说什么、说到什么程度”。

版式可以统一品牌感，但不该压过判断力。

真正好的自动化提醒，应该让人感觉：它知道今天有什么不同，也知道什么时候别打扰。

这比每天准时送来一张漂亮但平庸的表，重要得多。
